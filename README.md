<h2>☆サービス概要</h2>
架空のECサービスの各画面になります。<br>
ユーザーが望むタイミングでボトル入りのコーヒーを注文できるというサービスです。<br>
現在フロントのみの実装となります。<br>
画面遷移をしないように設計しSPA風に作成しました。<br>

<h2>☆実装機能</h2>
商品表示関連<br>
・一覧表示機能<br>
・詳細モーダル表示機能<br>
・件数表示機能<br>  
・しぼりこみ機能<br>
・検索機能<br>
・カート追加フィードバック機能(カートの機能はありません)<br>
<br>
